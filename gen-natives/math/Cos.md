# Cos
Takes a real value input in radians and returns its cosine value. The domain ofthe input is all real numbers and the range of the output is -1 to 1 inclusive.

## Declaration

```jass
native Cos takes real radians returns real
```

## Parameters
radians `real`
: The input radians.
