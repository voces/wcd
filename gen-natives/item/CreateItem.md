# CreateItem
Creates an item object at the specified coordinates ( x , y ).

## Declaration

```jass
native CreateItem takes integer itemid, real x, real y returns item
```

## Parameters
itemid `integer`
: The rawcode of the item.

x `real`
: The x-coordinate of the item

y `real`
: The y-coordinate of the item
